## Инструкция по установке

(Важно чтобы был установлен "Google Chrome")

1. Просто кидаешь в папку в годоте res://addons/  (res:// это просто файловая система внутри Godot)

 1.1 Лучше если полный путь аддона будет res://addons/Godot_agent/godot_agent/

2. Далее просто запускаешь плагин Проект -> Настройки Проекта -> Модули -> Gemini Agent (поставить галочку)

3. Агент будет находиться справа в последней вкладке

4. Нажимаешь новый чат и идет подключение к локальному серверу

4.1  После подключения к серверу появиться браузер

5. Нажимаешь на "Google ai studio" и на сайте гугл вводишь логин/пароль (пока не получилось синхронизировать информацию у гугл хрома, вероятно получиться у других браузеров в будущих обновлениях)

6. Теперь можно писать запросы , работает как обычный агент , умеет изменять файлы , создавать новые , перемещать (в целях безопасности удаление файлов пока не добавлял , скоро появиться)
